## Project Overview

An educational keylogger implementation created for learning about input capture techniques in digital forensics. This tool is strictly for educational and authorized security research purposes.

### Features

- Keystroke capture mechanism demonstration
- Log file management
- Timestamp-based recording
- Educational documentation

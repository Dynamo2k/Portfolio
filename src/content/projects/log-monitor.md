## Project Overview

A lightweight, shell-based log monitoring system that provides real-time analysis and alerting for security events across Linux systems.

### Features

- Real-time log file monitoring with inotify
- Pattern-based alert rules
- Email and webhook notifications
- Log rotation handling
- Multi-file monitoring support

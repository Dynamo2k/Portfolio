## Project Overview

FEAS is a comprehensive web-based faculty evaluation and feedback system designed for academic institutions to streamline the evaluation process.

### Features

- Online evaluation form submission
- Anonymous feedback system
- Statistical analysis and reporting
- Admin dashboard for managing evaluations
- Export functionality for reports

## Project Overview

SecureDork automates Google dorking for reconnaissance during penetration testing engagements, helping security professionals discover exposed sensitive information.

### Key Features

- **Pre-built Dork Templates**: 100+ categorized Google dork queries
- **Custom Query Builder**: Create and save custom dork patterns
- **Result Parsing**: Automated extraction of relevant findings
- **Export Reports**: Generate professional reports from findings
- **GUI Interface**: User-friendly interface for non-technical users

### Categories

- File exposure detection (PDFs, configs, databases)
- Login page discovery
- Sensitive directory enumeration
- Technology fingerprinting
- Vulnerability-specific queries
